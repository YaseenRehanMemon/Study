How to Study Each Block

Block 1: Finish Ch13 Rules (13.6 + Ex 13.2)

  • Skip all proofs. You already know first principle from 13.1-13.5. Now just learn the shortcut rules.
  • Write these 6 formulas on a separate paper like a cheat sheet:
    1) d/dx(c) = 0   2) d/dx(x^n) = nx^(n-1)   3) d/dx[cf] = cf'
    4) d/dx[f+g] = f'+g'   5) (fg)' = f'g + fg'   6) (f/g)' = (f'g - fg')/g^2
  • Read Examples 7, 9, 10, 11 from the book. Cover the solution, try it yourself first, then check.
  • Do Ex 13.2: Q1 (i, iii, vi, vii), Q2, Q4. Skip Q1(x, xi) for now - those are hard and low-yield.
  • Speed trick: For product/quotient rule, always label f and g first, find f' and g' separately, then plug into formula.

Block 2: Ch13 Applications + Ch14 Basics

  • 13.7 is easy - just "find derivative then plug in value". Read Examples 12-13 only. Skip 14-15 (investment/stress are low-yield).
  • Do Ex 13.3: Q1 only. Q2 is similar pattern.
  • Then start Ch14. 14.1 is mostly recall from 1st year. Skim quickly:
  • Focus on NEW things: R^3 vectors [x,y,z], magnitude = sqrt(x^2+y^2+z^2), unit vector = u/|u|
  • Direction cosines: cos a = x/r, cos b = y/r, cos c = z/r. Just remember: component divided by magnitude.
  • Don't waste time on proofs of vector properties (pg 263-264). Just know they exist.

Block 3: Dot Product Mastery (14.2 + Ex 14.2)

  • This is the MOST EXAM-HEAVY section of Ch14. Master this block.
  • Key formulas to memorize:
    u.v = a1*a2 + b1*b2 + c1*c2 (component form)
    u.v = |u||v| cos theta (angle form)
    cos theta = u.v / (|u||v|)
    Projection of v along u = u.v / |u|
    Work = F . d
  • Orthogonal means dot product = 0. This is asked in every exam.
  • Do Ex 14.2: Q1(i), Q3, Q4(i), Q5(i), Q6, Q13, Q14. These cover all types.
  • Read Examples 7 (cosine law proof), 12 (angle), 14 (work done).

Block 4: Cross Product + Exercises + Recap

  • Cross product = determinant of 3x3 matrix with i,j,k on top. This is pure mechanical computation.
  • Key formula: u x v = |i j k; a1 b1 c1; a2 b2 c2| (expand along first row)
  • Remember: i.i=j.j=k.k=0 (parallel cross = 0), ixj=k, jxk=i, kxi=j
  • Read Examples 15, 16. Do the cross product computations yourself.
  • Go back and do Ex 14.1: Q1(i), Q2(i), Q4, Q5, Q11(i). These are quick.
  • Last 10 min: Write ALL formulas from Ch13+Ch14 on one page. This is your revision sheet for exam day.
